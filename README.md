### üîß **Configuring "Add App" Button**
To make the "Add App" button work for servers directly:
1.  Go to the **Discord Developer Portal**.
2.  Select your App -> **Installation**.
3.  Under **Installation Contexts**, check **Guild Install**.
4.  **CRITICAL FIX**: Under **Default Install Settings**, select **Guild Install**.
    -   Click the **Scopes** dropdown.
    -   Check `bot` (This is required for the bot to join the member list!).
    -   Check `applications.commands`.
    -   **Permissions**:
        -   To keep it simple: Check **Administrator** (easiest for small bots).
        -   Or check: **Send Messages**, **Embed Links**, **Attach Files**, **Manage Messages** (for dashboard cleanup), **Manage Events** (for sync).
5.  Save Changes.
